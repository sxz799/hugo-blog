什么是GMP模型？

  • G：gorotine（协程）
  • M：machine（内核线程）
  • P：processor(调度器)

Go语言运行时，通过核心元素G，M，P 和 自己的调度器，实现了自己的并发线程模型。调度器通过对G，M，P的调度实现了两级线程模型中操作系统内核之外的调度任务。
