声明channel

channel是go语言中的一种数据类型，也叫通道 声明方式

ch:=make(chan string, n)
  • ch : channel的变量名
  • chan : 声明channel的关键字
  • string : channel中存储额数据类型
  • n: 缓冲长度(不填时代表无缓冲)
